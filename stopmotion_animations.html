stopmotion Animation

These animations use several different animation forms, from Claymation to cut-out animations. Claymation consist of using clay to make characters and sets. these can be manipulated to make small movements. camera techniques used in live action can be used in these animations because the sets being 3d. Cut-out animation uses it uses pieces of flat paper that are shaped into characters. Making the characters Move in the animation is done two different ways. one-way requires each pose in the movement to have a different piece of paper. pivot points are used so that the characters can be manipulated into different poses. All the stop motion forms use the same basic techniques for creating the animations. taking a photo every time, the character is moved. then combining all the images into a video were each image lasts for half a second or less.
